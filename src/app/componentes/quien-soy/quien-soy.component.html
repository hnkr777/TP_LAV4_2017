Sala de juegos

Este sitio es un trabajo práctico, orientado en la experiencia de usuario y el manejo de las tecnologías de front-end, entre las cuales se encuentran Twitter Bootstrap, JQuery, Angular 2, Typescript, HTML5, CSS3, PHP, MySQL, javascript entre otras.
